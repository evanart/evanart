== Description ==

Override your titles, descriptions and keywords with page/post level input for greater control of **SEO**.

*SEO Cheese Features*

* Modify meta-deta for each page or post.
* Auto Generate your titles, descriptions and keywords when overrides are not present. ( Optional )
* Help you prevent duplication content issues.
* Automatically appends the tags as keywords when available.
* Makes sure keyword(s) are unique.

You can learn more about seo cheese on our **[website](http://seocheese.blog.co.uk/)** website.

== Installation ==

1. Unzip into your `/wp-content/plugins/` directory and make sure the files are inside the seo-cheese/ folder.
1. Activate the plugin through the Wordpress Panel -> 'Plugins'
1. Visit (*Settings - Seo Cheese*)
1. That's it.
