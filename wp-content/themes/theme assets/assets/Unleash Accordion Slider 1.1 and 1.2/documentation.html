Getting Started:

First, in your head section, you need to include 'jQuery', 'jquery.unleash.min.js', 'unleash.css' and in case you want to use special easing methods include Lemonsanver Easing Plugin and if you will use 'rotate' animation include zachstronaut plugin which have 2 js files.

Then in your body add a code that looks like this with your own classes names

Finally, in the head section trigger your plugin, using $(window).load() function avoid using $(document).ready()

$(window).load(function() {
$('#outer').unleash();
        });

To add your own options; here are the list of options that you can change:

$(window).load(function() {
        $('#outer').unleash({ //make sure that $('#outer') is the same as your main html class.
			   duration: 700, //The speed of the slide.
               childClassName: '.box', //The class name of the slide contents.
               captionClassName: '.caption_1', //The class name of the caption.
               SliderWidth: '960px', //Width of the whole slider.
               SliderHeight: '300px', //Height of the whole slider.
               width: 600, //Width of the selected slide. (without px)
               Event: "hover", //Event for expanding slide (hover or click).
               easing: "quadEaseOut", //Easing for slide.
               captionEasing: "backEaseInOut", //Easing for caption.
               CollapseOnMouseLeave: true, //Leave the last selected slide open when the mouse leaves or not.
               CaptionAnimation: "opacity" //Caption animation type (pop-up, opacity or rotate).
			});
    });

And you can style the captions as you like from the css file :)
